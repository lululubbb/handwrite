# 智能手写笔记转录系统 - 毕业设计完整项目

## 项目概述

本系统是一个完整的前后端分离的智能手写笔记转录系统，专为毕业设计开发，具备完整的用户管理、图像上传、OCR识别、排版还原等功能。

## 技术栈

### 前端技术
- **Vue 3** - 前端框架
- **Vite** - 构建工具
- **Element Plus** - UI组件库
- **Axios** - HTTP请求库
- **Vue Router** - 路由管理
- **Pinia** - 状态管理

### 后端技术
- **Flask** - Web框架
- **Flask-CORS** - 跨域支持
- **Flask-SQLAlchemy** - ORM框架
- **PaddleOCR-VL** - OCR识别模型
- **OpenCV** - 图像处理
- **NumPy** - 数值计算

### 数据库
- **SQLite** - 轻量级数据库

## 安装和运行

### 环境要求
- Python 3.8+
- Node.js 18+
- pip（Python包管理器）
- npm（Node.js包管理器）

### 安装步骤
conda create -n handwritte python=3.9 -y
conda activate handwritte
#### 1. 安装Python依赖
在根目录下：

```bash
pip install -r requirements.txt
```


#### 2. 安装前端依赖

```bash
cd frontend
npm install
```

### 运行项目

#### 方式一：使用启动脚本（推荐）

1. 双击 `backend/start_backend.bat` 启动后端
2. 双击 `frontend/start_frontend.bat` 启动前端

#### 方式二：命令行启动

**启动后端服务：**
```bash
cd backend
python app.py
```
**配置数据库**

**启动前端服务：**
```bash
cd frontend
npm run dev
```

### 访问系统

- 前端地址：http://localhost:5173
- 后端API：http://localhost:5000/api

## 功能模块详解

### 1. 用户管理模块

#### 功能特性
- **用户注册**：支持邮箱验证，密码加密存储
- **用户登录**：支持用户名/邮箱登录，密码验证
- **个人信息管理**：修改邮箱、头像
- **密码修改**：原密码验证，新密码强度检查
- **历史记录查询**：分页查询个人识别记录

#### API接口
- `POST /api/user/register` - 用户注册
- `POST /api/user/login` - 用户登录
- `GET /api/user/profile` - 获取用户信息
- `PUT /api/user/profile` - 更新用户信息
- `PUT /api/user/change-password` - 修改密码
- `GET /api/user/history` - 获取历史记录
- `GET /api/user/history/:id` - 获取记录详情

### 2. 图像上传与预处理模块

#### 功能特性
- **图片上传**：支持JPG/PNG格式，最大16MB
- **拖拽上传**：支持拖拽上传
- **图像预处理**：
  - 亮度调整
  - 清晰度增强
  - 倾斜校正
- **实时预览**：上传后实时预览

#### API接口
- `POST /api/upload/preprocess` - 图像预处理
- `GET /api/upload/preview/:filename` - 获取预览
- `POST /api/upload/save-record` - 保存识别记录

### 3. OCR识别模块（核心）

#### 功能特性
- **PaddleOCR-VL集成**：严格遵循官方教程
- **中文手写识别**：高精度识别中文手写文字
- **识别质量评估**：返回置信度评分
- **位置坐标返回**：返回每个文字的位置坐标
- **自动排版还原**：根据布局自动还原段落结构

#### API接口
- `POST /api/ocr/process` - OCR识别
- `GET /api/ocr/health` - OCR服务健康检查
- `POST /api/ocr/batch-process` - 批量识别

### 4. 排版还原模块

#### 功能特性
- **段落还原**：根据垂直间距自动分段
- **标题识别**：识别较短且置信度高的文本作为标题
- **列表识别**：识别带项目符号的列表项
- **格式化输出**：返回格式化的文本

### 5. 管理员后台

#### 功能特性
- **系统统计**：用户数、上传次数、识别字符数
- **今日/本周统计**：实时统计信息
- **活跃用户TOP 10**：按上传次数排序
- **用户管理**：启用/禁用/删除用户
- **记录管理**：查看/删除历史记录

#### API接口
- `GET /api/admin/stats` - 获取系统统计
- `GET /api/admin/users` - 获取用户列表
- `PUT /api/admin/users/:id` - 更新用户状态
- `DELETE /api/admin/users/:id` - 删除用户
- `GET /api/admin/records` - 获取记录列表
- `DELETE /api/admin/records/:id` - 删除记录

## 数据库设计

### 用户表 (users)
- id: 用户ID
- username: 用户名
- email: 邮箱
- password_hash: 密码哈希
- avatar: 头像
- role: 角色（user/admin）
- status: 状态（active/inactive）
- created_at: 创建时间
- updated_at: 更新时间
- last_login: 最后登录时间
- total_uploads: 总上传次数
- total_characters: 总识别字符数

### 历史记录表 (history_records)
- id: 记录ID
- user_id: 用户ID
- original_filename: 原始文件名
- processed_filename: 处理后文件名
- upload_time: 上传时间
- ocr_result: OCR识别结果（JSON）
- formatted_text: 排版还原后的文本
- character_count: 识别字符数
- confidence: 平均置信度
- processing_time: 处理耗时
- status: 状态（processing/completed/failed）
- notes: 备注

1. **首次运行**：会自动创建数据库文件和必要目录
2. **PaddleOCR模型**：首次运行会自动下载，需要联网
3. **文件大小限制**：上传文件最大16MB
4. **端口占用**：确保5000端口（后端）和5173端口（前端）未被占用
5. **Python版本**：建议使用Python 3.8-3.11
6. **Node.js版本**：建议使用Node.js 18+

## 常见问题解决

### Q: PaddleOCR安装失败
A: 使用国内镜像源安装：
```bash
pip install PaddleOCR -i https://mirror.baidu.com/pypi/simple
```

### Q: 前端无法连接后端
A: 检查后端服务是否启动，浏览器控制台查看跨域错误

### Q: 图片识别效果不佳
A: 确保图片清晰度足够，手写文字尽量工整，背景简洁

### Q: 数据库文件找不到
A: 数据库文件位于 `backend/database/ocr_system.db`

### Q: 上传文件失败
A: 检查文件大小是否超过16MB，文件格式是否正确
